Logging

All logging is done by Log4j.

In the [TWEIAL_HOME]/conf/ directory, the log4j.properties file is how you configure the logging.

Note:
This configuration file is AUTOMATICALLY RELOADED if changed. Therefore you can change the logging configuration of the application while the application is running. No need to stop and restart the application to pick up the changes.
