Scheduling to Run

Windows

Use the Task Scheduler to schedule running the ferris-tweial-1.1.6.0-windows-x86-64-jre/bin/start-tweial.bat file. Microsoft has a Schedule a task page with all the details.

Linux

Use CRON to schedule the running of the application. 

# Run Tweial email every 10 minutes
0,10,20,30,40,50 * * * * /home/username/ferris-tweial-1.1.6.0-linux-x86-64-jre/bin/start-tweial.sh > /dev/null

Happy Tweialing!
